Catalogs of Discrete Objects
============================

The main interface for dealing with data in the form of catalogs of discrete
objects is provided by subclasses of the
:class:`nbodykit.base.catalog.CatalogSource` object. Here, we provide an
overview of the class and details about how to load data from disk and
manipulate the data once loaded.
